# Prompt: Bugfix

## Goal
Fix a bug with the smallest safe change.

## Inputs
- Bug description:
- Steps to reproduce:
- Expected behavior:
- Actual behavior:
- Logs / error messages:
- Scope constraints:

## Instructions
1. Hypothesize root cause(s).
2. Propose the smallest fix.
3. Add/adjust tests if possible.
4. Provide a validation plan.
5. Update `todo.md` notes (bug, fix, validation).

## Guardrails
- Keep the patch minimal.
- Avoid broad refactors unless necessary.
